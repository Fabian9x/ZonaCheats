# ZonaCheats - Página Web Minimalista

Página web profesional y minimalista con partículas interactivas en el fondo, diseñada con React y configurada para GitHub Pages.

## 🎨 Características

- **Diseño Minimalista**: Paleta de colores rojo, negro y blanco
- **Partículas Interactivas**: Fondo animado con particles.js
- **React + Vite**: Stack moderno y rápido
- **GitHub Pages Ready**: Configurado para despliegue automático

## 🚀 Instalación

1. Instala las dependencias:
```bash
npm install
```

2. Ejecuta el servidor de desarrollo:
```bash
npm run dev
```

3. Construye para producción:
```bash
npm run build
```

## 📦 Despliegue en GitHub Pages

1. Asegúrate de que el repositorio se llama `ZonaCheats` (o actualiza `base` en `vite.config.js`)

2. Instala gh-pages si no lo tienes:
```bash
npm install --save-dev gh-pages
```

3. Despliega:
```bash
npm run deploy
```

4. Ve a la configuración de tu repositorio en GitHub:
   - Settings → Pages
   - Source: selecciona la rama `gh-pages`
   - Tu sitio estará disponible en: `https://[tu-usuario].github.io/ZonaCheats/`

## 🛠️ Tecnologías

- React 18
- Vite
- particles.js
- CSS3
